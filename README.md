# MusicMindAI
MusicMind is an iOS app that helps users manage and understand their music taste across multiple streaming services. The app connects Yandex Music, SoundCloud, and VK, analyzes listening preferences with AI, generates personalized playlists from text prompts, and supports track migration/export workflows.

MusicMind - iOS-приложение на SwiftUI, которое объединяет музыку из разных сервисов (Яндекс.Музыка, SoundCloud, VK) в одном интерфейсе и добавляет AI-слой для анализа вкуса.
Главная ценность — помочь пользователю понять свой музыкальный профиль, получить персональные рекомендации, собрать плейлист по текстовому запросу и упростить перенос треков между сервисами.

MusicMind — это мой пет-проект iOS-приложения на SwiftUI для работы с музыкой из разных сервисов в одном месте.
Проект сделан как портфолио-практика по SwiftUI, работе с API и AI-интеграциям.

### Что уже реализовано:
- подключение Яндекс.Музыки, SoundCloud, Spotify и VK;
- AI-анализ музыкального вкуса и треков;
- создание плейлистов по текстовому запросу;
- миграция треков в Яндекс.Музыку, SoundCloud, Spotify и VK;
- экспорт треков в CSV.
